CSS

  • 기본적으로 태그마다 h1--h6,p,hr 등의 태그들이 갖는 스타일이 있다.
  • 기본적인 스타일을 사용하고 추가로 스타일 속성을 정하거나 설정된 값을 변경할 수 있다.
    태그의 Style 속성값으로 설정 - 내용이 많아지면 관리와 가독성이 많이 떨어진다.
    따라서 다음과 같은 방법으로 설정하면 조금 더 관리하기 편리하게 세팅할 수 있다.
    1. Head안에 Style 태그로 설정
    2. 별도의 CSS파일을 만들어 활용 - head안에 link라는 태그로 연결.
