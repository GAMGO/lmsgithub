전역 객체

객체는 여러가지 데이터를 저장하는 형식임.

데이터를 구별하는 이름이 필요한데 다음과 같다 : 속성, 프로퍼티(property)

브라우저의 전역 객체는 여러 속상을 갖는다.
        - 전역 객체는 브라우저를 제어하는 기능도 가지고 있는데 이것을 메소드(Method)라고 부른다.
        - 객체의 구성 요소 : 속성과 메소드
