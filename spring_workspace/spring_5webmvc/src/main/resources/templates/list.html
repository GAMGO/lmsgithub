파라미터

터미널의 로그만 확인해주세요.
http://localhost:8081/list?name=tom&age=23 는 정상출력
http://localhost:8081/list?name=tom 은 오류. age는 int로 저장하는데 그 때 내부적으로 Integer.parseInt(문자열) 실행. 파라미터 값은 문자열입니다.

파라미터가 많은 경우 Query로 계속 연결됨, 파라미터 갯수 만큼 변수를 선언해야하고 너무 많다면 비효율적입니다.
따라서 DTO 클래스를 사용합니다. 스프링 부트는 파라미터 이름과 dto 변수명으로 매핑하여 저장합니다.
눌러라
http://localhost:8081/listB

POST 요청 처리 예시

파일전송

application.properties 파일에 서버에서 업로드 요청이 온 파일을 어느 폴더에 저장할 것인지 설정이 필요합니다.
Form태그의 필수 속성은 "method = "POST" enctype="multipart/form-data" 입니다.
컨트롤러 클래스에서는 파일을 MultipartFile 타입으로 저장해야 합니다.
