customer/list

애플리케이션의 기능에 따라 url 경로와 view 파일이 저자오디는 폴더를 추가할 수 있습니다. 여기서는 customer를 경로에 추가하고 view파일도 customer 폴더에 저장했습니다.

  • 애트리뷰트 이름 page로 저장된 데이터 출력
  • 애트리뷰트 이름 dto로 저장된 객체의 name필드 출력
  • 애트리뷰트 이름 dto로 저장된 객체의 address필드 출력
  • 애트리뷰트 이름 list로 저장된 객체 출력
Customer Attribute는 값이 없습니다.
